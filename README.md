## About This Project

Welcome to my Agile Project Management portfolio.

The project demonstrates how Agile frameworks, Jira, and Confluence were used to manage software development from project initiation through planning, sprint execution, release, and continuous improvement.

# Business Problem

Many professionals struggle to accurately track working hours, organise projects, and monitor productivity.

Manual processes often lead to:

- inaccurate reporting
- missed deadlines
- inefficient project tracking
- reduced productivity

FocusFlow was developed to solve these challenges by providing an intuitive web application for time tracking, project management, and productivity reporting.
