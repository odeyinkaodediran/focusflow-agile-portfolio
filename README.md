# FocusFlow MVP

## Agile Project Management Portfolio.

> Delivering software projects using Scrum, Kanban, Jira and Confluence.

---

## Project Summary

FocusFlow is a cloud-based project management application developed as an Agile MVP.

The project demonstrates end-to-end Agile delivery, including project planning, backlog management, sprint execution, stakeholder engagement, risk management, release planning, and continuous improvement.
